# crypto-price-checker
My masterpiece
A simple website to check cryptocurrency prices using a public API .
Built with **HTML**,**CSS** and **Javascript**.
##Features
* Real-time crypto price lookup
* Modern responsive design
* Clean card layout

##How to use
1. Open 'index.html' in your browser.
2. Enter the cryptocurrency symbol (e.g.,BTC,ETH).
3. Click **Check Price** to see the current value.
